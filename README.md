# 🛡️ The Leak App

## Application Vulnérable pour la Formation en Audit de Sécurité Assisté par IA

Bienvenue dans "The Leak App", une application Node.js volontairement vulnérable conçue pour vous aider à pratiquer l'audit de sécurité assisté par intelligence artificielle. Ce projet est utilisé dans le cadre d'un TP pour identifier, analyser et corriger des failles de sécurité critiques.

---

### 🚀 Démarrage Rapide

Pour lancer l'application et commencer le TP, suivez ces étapes simples :

1.  **Prérequis :**
    Assurez-vous que [Docker](https://www.docker.com/get-started/) et [Docker Compose](https://docs.docker.com/compose/install/) sont installés sur votre machine.

2.  **Lancement de l'application :**
    Ouvrez votre terminal, naviguez jusqu'à la racine de ce projet (`leak-app/`), puis exécutez la commande suivante :

    ```bash
    docker-compose up --build
    ```

    Cette commande va construire les images Docker nécessaires et démarrer l'application.

3.  **Accès à l'application :**
    Une fois le serveur lancé, accédez à l'interface de l'application via votre navigateur web à l'adresse :
    http://localhost:3000
